🚀 QUICK START
================================================================================

Setup (5 minutes):
  python3 -m venv .venv
  source .venv/bin/activate
  pip install -r requirements.txt
  cp .env.example .env
  nano .env  # Add your device UDID

Run Tests (2 terminals):
  Terminal 1: appium --port 4723
  Terminal 2: pytest tests/ -v
  open reports/report.html
